# African Weather Analysis

## Project Overview
This project analyzes weather patterns across African countries using Python. The analysis focuses on temperature, rainfall, air quality, and regional climate differences.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Dataset
The dataset contains weather-related features such as temperature, humidity, wind speed, precipitation, and air quality indicators for locations across Africa.

## Key Analysis Areas
- Temperature trends
- Rainfall patterns
- Air quality analysis
- Regional weather comparison

## Conclusion
The project helps identify climate patterns and regional differences in African weather data through data cleaning, analysis, and visualization.
